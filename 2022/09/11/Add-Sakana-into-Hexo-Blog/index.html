没有石蒜第12话看我要死了

插件概述

插件 GitHub 地址：MingxuanGame/hexo-sakana

此 Hexo 插件用于在渲染 HTML 时将 Sakana! Widget 组件注入进 body 内，以非侵入式方式加载石蒜组件。

注入代码参考 EYHN/hexo-helper-live2d。

直接cv readme

安装

直接 npm
