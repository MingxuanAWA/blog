驱动器、适配器和插件

NoneBot2 的组成主要有以下三部分：

由 NoneBot2 使用：驱动器（Driver）、适配器（Adapter）

由用户扩展：插件（Plugin）

它们的关系以及交互流程如下

说白了，NoneBot2就是一个中间商

Driver

驱动器类型分为两类

1.ForwardDriver 客户端类型驱动器
2.ReverseDriver 服务端类型驱动器

TIP：两种驱动器可同时使用

ForwardDriver

ForwardDriver 用于发送 HTTP 请求和建立正向 WebSocket 连接

有以下驱动器：

1.HTTPX（仅支持HTTP请求）
2.websockets（仅支持WebSocket 连接）
3.aiohttp

ReverseDriver

ReverseDriver 用于建立 HTTP WebHook 服务器和反向 WebSocket 连接

有以下驱动器：

1.FastAPI（默认）
2.Quart
