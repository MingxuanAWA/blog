NoneBot2 新手 FAQ

这里整理了一些新手经常提问的问题，这些问题往往很简单（文档里均有相关内容），但是可能因为某些原因，会导致下面的错误

下面将根据如下常常提问的问题进行解答

我要快速精通 Python

洗洗睡吧，学习都是循序渐进的

你要是能 3 天精通 Python，为什么不去找工作呢

有没有通俗易懂的 NoneBot 教程

Well404的教程

Bad Handshake(403)

未注册协议适配器

相关文档：使用适配器、配置连接

在 bot.py 添加如下代码

记得下次在脚手架选中适配器

TypeError: Driver.register_adapter() takes 2 positional arguments but 3 were given

NoneBot2 beta-1 版本修改了注册适配器的方式，需要传入适配器 Adapter

若想使用老版本请在安装时指定

注册适配器参考上面

nb: 命令未找到

可能是未安装脚手架，或脚手架不在Path中

相关文档：使用脚手架

可使用 python3 -m nb_cli 代替

如何发送图片

发送图片以 OneBot 协议适配器为演示

麻烦不要使用 CQ 码了

相关文档：处理消息、MessageSegment-image

使用 MessageSegment.image 构造图片消息段

支持 Path、str、BytesIO 和 bytes 类型

ValidationError

未填写插件的配置

请在 .env、.env.dev 或 .env.prod 填写插件的配置

配置参考插件的文档

NotImplementedError 且 Traceback 出现 asyncio 字样

未关闭重载

相关文档：选择驱动器#fastapi_reload

在配置关闭 fastapi_reload 即可

也可参考nonebot/nonebot2#376#issuecomment-846523618的猴子补丁修复

详情参考nonebot/nonebot2#376、nonebot/nonebot2#830

账号被风控

tx不开心，把你的机器人风控了（我们称之为喜报）

目前可能解除风控的办法：

1.切换协议为 MacOS
2.使用企业QQ（你不一定能买得起）
3.收购腾讯

Check your plugin name

插件重复加载，检查 bot.py 和 pyproject.toml 是否已经加载（比如已经在 pyproject.toml 如下设置，请删除 bot.py 的加载代码）

具体情况因配置而异

为什么没有 XX 功能

你去提 pr 就有了

最后

如果这个文章无法帮到你，请多翻阅文档，那里可能有你想要的答案

也可通过这些方式提问
